Display components for presenting content and creating layouts. These are primarily presentational; state management is optional for components like modals and toasts.

## Summary

- accordion.templ — Collapsible section component for grouping content
- alert.templ — Alert/notification box with title, description, and icon variants (success, warning, error, info)
- cloakLoading.templ — Overlay to hide content while loading
- loadingOverlay.templ — Full-screen overlay with loading indicator
- modal.templ — Dialog/modal component with header, content, footer
- sidebar.templ — Vertical navigation panel with collapsible sections
- tag.templ — Badge/label component for categorization
- toast.templ — Dismissible notification component with Alpine.js state management
- toastState.js — Helper JavaScript for managing toast notifications

## Constraints

- MUST support nested templ.Component for content composition
- MUST use Phosphor icons for visual elements
- MUST support various size and styling variants

## Guidance

- SHOULD support Alpine.js for interactive components (modal visibility, toast dismissal)
- SHOULD include optional left and right icons for alerts and badges
- SHOULD provide both HTML and state-based binding options for content
