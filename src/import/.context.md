Export layer for third-party dependencies and shared resources. Provides templ components that embed CDN links and bundled utilities as head tags and inline scripts.

## Summary

- import.templ — Templ components exporting CDN links for Alpine.js, Tailwind CSS, Phosphor Icons, Google Fonts, HTMX, and the UI toolset
- toolset.js — JavaScript utility library bundled and embedded via import.templ

## Constraints

- MUST use SRI (Subresource Integrity) hashes for all external CDN resources
- MUST support two variants: HeadTagsMinimal() and HeadTagsFull()
- MUST export the toolset.js inline via MinifierTemplateJs()

## Guidance

- SHOULD pin CDN resources to specific versions
- SHOULD provide modular head tag components (e.g., HeadTagsAlpineJs, HeadTagsHtmx) for selective inclusion
- SHOULD minify embedded JavaScript before rendering
