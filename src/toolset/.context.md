Go-side utilities for code generation and content transformation. Uses esbuild to minify JavaScript and CSS at compile time or runtime.

## Summary

- minifier.go — Wrapper around esbuild API for minifying JavaScript and CSS content, with options for whitespace, identifiers, and syntax minification
- minifier_test.go — Tests for the minifier utility

## Constraints

- MUST support both JavaScript and CSS content types
- MUST gracefully fall back to unminified content if esbuild errors occur
- MUST return pointers to strings for memory efficiency

## Guidance

- SHOULD provide convenience functions (MinifierJs, MinifierCss) with sensible defaults
- SHOULD offer templ.Component wrapper functions for inline rendering
- SHOULD support configurable minification settings via MinifierSettings struct
