Interactive control components that respond to user events. Components in this directory have state management capabilities and trigger handlers on user interaction.

## Summary

- button.templ — Generic button component with customizable styling, icons, and click handlers
- rangeSlider.templ — Input range slider with min/max constraints, step values, and Alpine.js state binding

## Constraints

- MUST support OnClickFunc and OnClickFuncName patterns for event handlers
- MUST support optional icons (left and right) using Phosphor icon names
- MUST support Alpine.js integration via TwoWayStatePath for slider values

## Guidance

- SHOULD include size and shape variants
- SHOULD support disabled state with visual feedback
- SHOULD include tooltip support where appropriate
