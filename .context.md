Infinite UI is a Go component library for building elegant user interfaces using templ templates, Alpine.js, Tailwind CSS, and Phosphor Icons. The library provides reusable, configurable components organized by function (form inputs, display elements, controls).

## Summary

- src/ — Reusable component library organized by category (control, display, form, import, toolset)
- demo/ — Static HTML demo generator showcasing all components
- go.mod — Main module definition (library only, no executable)
- ui.go — Root package file (empty main function for compatibility)
- tailwind.config.js — Tailwind CSS configuration with custom primary/secondary colors
- README.md — Project documentation and usage guide
- CHANGELOG.md — Version history
- LICENSE.md — MIT license
- docs/ — Static documentation directory (empty in current state)
- .agents/ — Agent framework files (personas, skills, rules)

## Constraints

- MUST NOT include Alpine.js, Tailwind CSS, or Phosphor Icons in the bundle; depend on CDN versions
- MUST NOT require HTMX (though designed to work well with it)
- MUST use the a-h/templ template engine for all UI composition
- MUST provide CDN imports via the import package for downstream consumers

## Guidance

- SHOULD maintain backward compatibility across minor versions
- SHOULD follow the settings-struct pattern consistently across all components
- SHOULD document component APIs in README.md and showcase them in demo/demo.templ
