Static HTML demo generation. Contains a templ-based showcase of all UI components with usage examples. The demo is built as a separate Go module to avoid circular imports.

## Summary

- demo.go — Entrypoint that renders DemoIndex() to index.html
- demo.templ — Template defining the full demo page structure, navigation sidebar, and all component usage examples
- demo_templ.go — Compiled templ output (auto-generated)
- go.mod — Separate Go module to isolate demo dependencies
- go.sum — Dependency checksums

## Constraints

- MUST be a separate Go module to prevent circular dependencies with the main ui package
- MUST generate a static index.html file (not served as a live endpoint)

## Guidance

- SHOULD showcase all components from src/ with realistic configurations
- SHOULD include navigation sidebar for easy browsing of component categories
- SHOULD demonstrate both basic and advanced usage patterns for each component
